Porquê este grupo?

Decidimos formar este grupo, pois achámos que seria uma ótima parceria para fazer este projeto.

Todos nós, desde sempre, gostámos de Pokémon, o que foi uma grande ajuda para desenvolver o website, pois para além de estarmos dentro do assunto, é algo ligado diretamente à nossa infância, o que nos entusiasmou e inspirou a fazer um bom trabalho.

Como foi fazer este Website:

Para fazer este website usámos principalmente três ferramentas, HTML5, CSS3 e JavaScript.

Foi um desafio transportar as ideia do papel para a realidade, porém aquilo que nos motivou mais, foi ver as alterações que íamos fazendo ganhar vida e ver o nosso website a evoluir constantemente. Foi extremamente satisfatório.

Todos nós esperamos ter mais oportunidades para desenvolver outros websites!
